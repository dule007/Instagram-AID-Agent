# Claude Code upute

Claude Code radi kao ravnopravan suradnik s Codexom u ovom repozitoriju.

Prije bilo kakve izmjene obavezno pročitaj cijeli `AGENTS.md`. On je kanonski izvor poslovnih, sigurnosnih i suradničkih pravila.

Zatim pročitaj:

1. `coordination/CURRENT_HANDOFF.md`;
2. `coordination/OWNERSHIP.md`;
3. `coordination/DECISIONS.md`.

## Claude Code početna procedura

1. Pokreni `git status --short --branch`.
2. Sažmi korisniku što je već napravljeno i što namjeravaš mijenjati.
3. Preuzmi ciljane datoteke u `coordination/OWNERSHIP.md`.
4. Ne mijenjaj datoteke koje trenutačno drži Codex ili drugi agent.
5. Nakon validacije ažuriraj `coordination/CURRENT_HANDOFF.md` i oslobodi ownership.

## Važna granica

Claude Code i Codex nemaju ugrađen izravni međusobni chat. Razmjenjuju stanje isključivo kroz Git, sadržaj repozitorija i datoteke u `coordination/`. Ne pretpostavljaj da je drugi agent vidio poruku iz zasebne sesije ako ona nije zabilježena u repozitoriju.

Ne koristi niti zapisuj Instagram lozinku. Ne objavljuj sadržaj i ne aktiviraj Meta, Canva ili ManyChat integracije bez izričitog korisničkog odobrenja i odgovarajuće službene autorizacije.
